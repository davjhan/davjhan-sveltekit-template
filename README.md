# About

This is a starter template for davjhan.com sveltekit projects.

This template uses

-   Sveltekit
-   TailwindCSS

# Setup Checklist

1. Update the project name in `package.json`
2. Update the page title in index.html
3. `pnpm install`

# Running

### Dev server

```bash
pnpm run dev
```

### Deploy

```bash
pnpm run deploy
```
